INSTALLATION:
-------------
Run this command to install:
    cd /home/bob/projects/photo-converter/family-media-manager
    sudo bash INSTALL.sh

Then:
1. Activate plugin in WordPress admin
2. Create two pages with shortcodes:
   - [fmm_registration]  (for family registration)
   - [fmm_media_library] (for video library)
3. Configure settings in Family Media > Settings
4. Start inviting family members!

HOW IT WORKS:
-------------
1. YOU (Admin):
   - Send invitation to family@email.com
   - They get email with registration link + code

2. FAMILY MEMBER:
   - Clicks link or visits registration page
   - Enters their info and password
   - Gets automatically logged in

3. VIEWING VIDEOS:
   - Family member logs in
   - Visits "Family Videos" page
   - Can search, filter, watch, or download
   - All access is tracked and secured

4. UPLOADING VIDEOS:
   - You upload videos via WordPress admin
   - Thumbnails generated automatically
   - Videos appear instantly to all family members
   - Protected by .htaccess and authentication
